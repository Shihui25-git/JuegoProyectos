ECO-PROYECT
-----
Proyecto educativo de desarrollo colaborativo para la creación de un videojuego web basado en microjuegos relacionados con los Objetivos de Desarrollo Sostenible (ODS).

Este proyecto ha sido desarrollado por tres alumnos del IES Abastos (Valencia) como parte de una práctica académica orientada a aprender el uso de herramientas de Inteligencia Artificial en el desarrollo de software, utilizando metodologías ágiles y control de versiones.

📖 Descripción
----

El proyecto consiste en el desarrollo de un videojuego web educativo compuesto por varios microjuegos de corta duración, cada uno relacionado con uno o varios Objetivos de Desarrollo Sostenible (ODS).

El jugador deberá superar diferentes desafíos en un tiempo limitado mientras avanza en la puntuación. Cada error reducirá las vidas disponibles, hasta que el juego termine.

Además del desarrollo técnico, el proyecto busca simular un entorno real de trabajo en equipo dentro del desarrollo de software.

🤖 Uso de Inteligencia Artificial
----

Este proyecto forma parte de una práctica destinada a aprender a utilizar herramientas de Inteligencia Artificial como apoyo en programación.

  La IA se ha utilizado para:

  Asistencia en la generación de código

  Resolución de problemas técnicos

  Apoyo en el diseño de funcionalidades

  Mejora de la estructura del proyecto

  Creación de documentación

El objetivo principal es aprender a integrar la IA como herramienta de apoyo dentro del flujo de trabajo de desarrollo.

🎯 Objetivos del Proyecto
----

Desarrollar un videojuego web educativo.

Implementar microjuegos interactivos.

Aplicar metodologías ágiles (Kanban).

Utilizar Git y GitHub para el trabajo colaborativo.

Aprender a utilizar herramientas de Inteligencia Artificial en el desarrollo de software.

Simular un entorno de desarrollo profesional en equipo.

🕹️ Mecánica del Juego
-----

El juego funciona mediante una serie de microjuegos encadenados.

Características principales:

Menú principal de acceso al juego

El jugador comienza con 4 vidas

Cada microjuego tiene un objetivo claro y duración corta

Cada fallo resta una vida

El juego termina al perder todas las vidas

La puntuación aumenta al completar microjuegos correctamente

🎲 Microjuegos
-----

Cada microjuego debe incluir:

🎯 Un objetivo claro

⏱️ Tiempo límite para completarlo

🖱️ Interacción mediante ratón y/o teclado

📈 Tres niveles de dificultad
